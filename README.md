# Installation

### Clone the repo
    ```bash
    git clone https://github.com/V1337Q/decay-vim.git
    ```

### cd to the cloned repo
    ```
    cd decay-vim
    ```

### Move the theme (.vim) file into ~/.vim/colors/ (If not exist, make one)
    ```
    cp -r decay.vim ~/.vim/colors/
    ```
### Set colorscheme on your .vimrc
    ```
    colorscheme decay
    ```
